# もみ徳｜口コミアシスタントツール

このツールは「もみ徳」様向けに開発された口コミ支援ツールです。  
HTML・CSS・JavaScriptで構成され、スマホに最適化されたシンプルなUIを提供します。

---

## 📦 含まれるファイル

- index.html：トップページ（性別選択）
- female.html / male.html：口コミ生成ツール（性別別UI）
- phrases.js：口コミ語録（トーン別語尾ランダム対応）
- 各種アイコン・ロゴ画像
